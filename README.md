# 🔔 Pop-Notify

Pop-Notify é um pacote de notificações reutilizáveis desenvolvido com JavaScript puro, HTML5 e CSS3, criado para facilitar a implementação de alertas e mensagens dinâmicas em aplicações web.

O projeto foi desenvolvido com foco em reutilização, personalização visual e experiência do usuário.

---

## 🚀 Funcionalidades

- ✅ Notificações de sucesso
- ❌ Alertas de erro
- ⚠️ Mensagens de aviso
- ℹ️ Notificações informativas
- 🎨 Personalização visual
- ✨ Animações suaves
- 📱 Layout responsivo
- 🔄 Controle dinâmico de exibição
- ♻️ Componentes reutilizáveis

---

## 🛠️ Tecnologias Utilizadas

- JavaScript
- HTML5
- CSS3
